# Soluciones de Acepta el Reto

Este repositorio contiene mis soluciones a diversos problemas de la plataforma Acepta el Reto.
El objetivo es practicar programación, mejorar lógica y documentar mi progreso resolviendo ejercicios de distinto nivel.

Cada carpeta o archivo corresponde a un problema concreto, incluyendo su solución implementada y enunciado.
